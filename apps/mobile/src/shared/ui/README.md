# UI

Status: shell only.

Purpose:
Reusable UI primitives such as buttons, cards, modals, and layout components shared across all NoteBox features.

Allowed:
- reusable, cross-feature code only

Forbidden:
- feature-specific business logic
- screen-specific state
- API behavior owned by one feature
- product doctrine changes
- prototype-only runtime behavior
- Vite/browser-only behavior
